# DiabloVault

DiabloVault is an offline, file-based item vault and inventory manager for
Diablo 1 and DevilutionX save files.

## Goals
- Safely organize items across characters and stash files
- Enforce Single Player / Multiplayer separation
- Provide fast, Explorer-style workflows (drag & drop, cut/paste)
- Support duplicate detection and cleanup

## Non-Goals
- No in-game memory editing
- No trainers or cheats
- No network features

## Status
Early development. Toolchain setup in progress.
